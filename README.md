# EMS (Employee Management System)

EMS is a .NET ASP.NET Core MVC web application built using Entity Framework Core and SQL Server. It demonstrates CRUD operations (Create, Read, Update, Delete) and follows MVC architecture.

## Technologies Used
ASP.NET Core MVC, Entity Framework Core, C#, SQL Server, HTML, CSS, Bootstrap.

## Project Structure
Controllers handle logic, Models represent database tables, Views handle UI, and DbContext manages database connection.

## How to Run
Clone repo: https://github.com/amitha-sadhoon/EMS.git  
Open in Visual Studio  
Restore NuGet packages  
Update connection string in appsettings.json  
Run migrations using "update-database"  
Run project using IIS Express or dotnet run

## Features
Add, view, edit, and delete records using Entity Framework Core with SQL Server integration.
